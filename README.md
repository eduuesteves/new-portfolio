## Para iniciar o projeto:

    pnpm create vite@latest
    |
    name: porfolio
    |
    cd portfolio
    |
    pnpm i
    |
    code . (se assim como eu, você utilizar o VS Code)

## Pacotes utilizados:

    🔴 sass
    🔵 react
    🟢 react-dom
    🟡 phosphor-react

Observação: Cada componente terá uma função e apenas!
